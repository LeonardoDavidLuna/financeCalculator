# FinanceCalculator - Tu Calculadora de Gastos Básica

[![Android](https://img.shields.io/badge/Android-green?style=for-the-badge&logo=android)](https://www.android.com/)

**FinanceCalculator** es una aplicación móvil desarrollada para Android que te permite llevar un registro de tus gastos diarios y mensuales de una manera sencilla y organizada. Olvídate de las hojas de cálculo complicadas, con FinanceCalculator tendrás toda tu información financiera al alcance de tu mano.

## Características Principales

* **Registro de Gastos Intuitivo:** Guarda tus gastos de forma rápida y fácil, indicando la fecha, el monto y el rubro al que pertenece.
* **Base de Datos Local con SQLite:** Toda tu información se almacena de forma segura en tu dispositivo, garantizando tu privacidad y acceso sin necesidad de conexión a internet.
* **Gestión de Usuarios:**
    * **Creación de Cuenta Local:** Regístrate de forma segura utilizando tu número de celular.
    * **Actualización de Datos:** Mantén tu información personal y saldo siempre actualizado.
    * **Consulta de Datos:** Visualiza tu información de perfil en cualquier momento.
    * **Eliminación de Cuenta:** Opción para eliminar tu cuenta y toda la información asociada de la aplicación.
* **Filtrado de Gastos:**
    * **Por Fecha:** Visualiza tus gastos en rangos de fechas específicos (diario, semanal, mensual, personalizado).
    * **Por Rubro:** Analiza en qué categorías estás gastando más gracias al filtrado por rubros (comida, transporte, entretenimiento, etc.).

## Tecnologías Utilizadas

* **Android SDK:** Plataforma de desarrollo nativo para Android.
* **Java:** Lenguaje de programación principal para el desarrollo de la aplicación.
* **SQLite:** Sistema de gestión de bases de datos relacional embebido para el almacenamiento local de datos.
* **Android Jetpack:** Conjunto de bibliotecas y herramientas para facilitar el desarrollo de aplicaciones Android robustas como Room Persistence Library para la interacción con SQLite.
* **Diseño de Interfaz de Usuario (UI):** XML para la estructura y diseño de las pantallas.

## Cómo Utilizar

1.  **Descarga e Instalación:** Descarga la aplicación desde aquí... próximamente.
2.  **Creación de Cuenta:** Al iniciar la aplicación por primera vez, se te pedirá crear una cuenta proporcionando los datos solicitados.
3.  **Registro de Gastos:** Utiliza el formulario para ingresar tus gastos diarios, seleccionando la fecha, el monto y el rubro correspondiente.
4.  **Visualización de Gastos:** Navega por las diferentes secciones para ver tus gastos diarios y mensuales.
5.  **Filtrado de Gastos:** Utiliza las opciones de filtrado por fecha y rubro para analizar tus patrones de gasto.
6.  **Gestión de Cuenta:** Accede a la sección de perfil para actualizar tus datos, consultar tu información o eliminar tu cuenta.
